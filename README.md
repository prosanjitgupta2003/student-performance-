# Student Performance Prediction using Decision Tree

## Description
This project predicts student performance (grade) based on academic behavior and engagement using a Decision Tree Machine Learning model.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Model Used
- Decision Tree Classifier

## Dataset Features
The model uses the following input features:
- Weekly Self Study Hours
- Attendance Percentage
- Class Participation
- Total Score

## Target Variable
- Grade

## Evaluation Metric
- Accuracy Score

## Results
- Decision Tree Accuracy: 99%

## Features of the Project
- Data preprocessing and cleaning
- Feature selection
- Model training and evaluation
- High accuracy classification

## Conclusion
The Decision Tree model achieved 99% accuracy in predicting student grades based on academic and behavioral features.
